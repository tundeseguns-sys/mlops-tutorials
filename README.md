# MLOps Tutorials

This repository is for learning Machine Learning Operations from beginner level to production level.

## Project Goal

The goal is to build a complete MLOps workflow step by step.

## Planned Workflow

1. Set up GitHub repository and project structure
2. Create and manage a Python virtual environment
3. Load and clean data
4. Train a machine learning model
5. Save the trained model
6. Track experiments
7. Test the code
8. Build a prediction API
9. Containerize the app with Docker
10. Deploy and monitor the model
